# Netflix Content Analytics Dashboard (Power BI)

## Project Overview
Interactive Power BI dashboard analyzing Netflix content trends, ratings, countries, and growth over time.

## Key Insights
- Content increased rapidly after 2016
- Movies dominate over TV Shows
- Top 10 countries contribute majority of content
- TV-MA is the most common rating

## Skills Demonstrated
- Data Cleaning (Power Query)
- DAX Measures
- Data Modeling
- Interactive Dashboards
- Data Visualization

## Tools Used
Power BI, DAX
